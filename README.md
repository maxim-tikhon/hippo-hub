## HippHub - Fullstack Marketplace for Digital Products

## Built with
- **Next.js**
- **React**
- **Typescript**
- **Tailwind CSS**
- **shadcn/ui**
- **Payload + mongodb**
- **tRPC**
- **Zustand**
